## Instalacion

Con node en su maquina, proceda a instalar las dependencias con el siguente comando

```bash
npm install
```
Luego ya instalada las dependencias ejecutamos el comando para iniciar la app , quedara expuesto en el puerto 3000

```bash
npm start
```
